🎮 Videogioco di Orest

Un piccolo videogioco creato con HTML, CSS e JavaScript.

Il gioco è stato sviluppato da Singer Orest come progetto creativo di programmazione.

🕹 Come si gioca

Muovi il personaggio con le frecce della tastiera

Oppure usa i pulsanti sullo schermo da cellulare

Raccogli le monete dorate

Ogni moneta aumenta il punteggio

⭐ Funzionalità del gioco

🎮 Movimento del personaggio

💰 Monete da raccogliere

📈 Sistema di punteggio

🏆 Record salvato

📊 Livelli di difficoltà

📱 Controlli per cellulare

🚀 Come avviare il gioco

Scarica il file gioco.html

Aprilo con un browser (Chrome, Edge, Firefox)

Inizia a giocare!

🌐 Gioca online

Puoi giocare direttamente dal browser:
